## О проекте

**BarberTime** — клиент-серверное React-приложение, в котором пользователь может ознакомиться с услугами барбершопа, выбрать дату и время посещения и оформить запись онлайн.

Записаться можно без регистрации. Авторизованному пользователю дополнительно доступны личный кабинет, сохранение сессии, автоматическое заполнение контактных данных и управление своими записями.

## Возможности

- выбор услуги в каталоге или форме записи;
- создание записи для гостя или авторизованного пользователя;
- регистрация и вход с сохранением сессии в `HttpOnly` cookie;
- восстановление авторизации после перезагрузки страницы;
- защищённый маршрут личного кабинета;
- просмотр статистики и истории записей;
- отмена предстоящих записей;
- валидация форм и отображение ошибок запросов.
- получение списка услуг через API;

## Технологии

- **React 19**, **TypeScript**, **Vite**;
- **React Router**;
- **Axios**;
- **React Hook Form** и **Yup**;
- **SCSS Modules**;
- **React Context** для состояния авторизации;
- **vite-plugin-json-server** и Vite middleware для эмуляции API;
- **JWT** в `HttpOnly` cookie и **bcryptjs** для хеширования паролей.

Интерфейс собран из собственных UI-компонентов без сторонних UI-библиотек.

## Запуск проекта

Для запуска понадобятся Node.js и npm.

```bash
npm install
npm run dev
```

После запуска приложение обычно доступно по адресу:

```text
http://localhost:5173
```

Дополнительная настройка `.env` и отдельный запуск сервера не требуются. Данные хранятся в `server/db.json`.

> Для полноценной работы интерфейса и API используйте `npm run dev`. Серверная эмуляция встроена в dev-сервер Vite и не запускается командой `npm run preview`.

## Команды

| Команда           | Назначение                                         |
| ----------------- | -------------------------------------------------- |
| `npm run dev`     | запуск приложения и эмуляции API                   |
| `npm run build`   | проверка TypeScript и production-сборка            |
| `npm run lint`    | проверка проекта ESLint                            |
| `npm run preview` | просмотр собранного клиентского приложения без API |

## Структура

```text
src/
├── Api/          # функции запросов к API
├── Components/   # Pages, Layouts, Widgets и UI-компоненты
├── Context/      # состояние и методы авторизации
├── Schemas/      # схемы валидации Yup
├── Types/        # общие TypeScript-типы
└── assets/       # изображения и иконки

server/
├── auth.ts             # JWT, cookie, пользователи и пароли
├── authPlugin.ts       # регистрация, вход, сессия и выход
├── bookingsPlugin.ts   # создание, получение и отмена записей
└── db.json             # файловое хранилище
```

Подробное описание серверной эмуляции и доступных маршрутов находится в [`server/README.md`](server/README.md).
